## Guide

Build a program based on Fuzzy Logic (Mamdani Inference System) according to the rules provided in [Exercise 18](../../docs/applied-computational-intelligence-exercise18.pdf) of the book "Computational Intelligence Applied to Administration, Economics and Engineering in Matlab", Hime Aguiar and Oliveira Junior (Coordinators), Caldeira, A. M.; Machado, M.A.S.; Sousa, R.C.; Tanscheit, R.; Thomson Learning, 2007. The program should prompt the user as input and display the output produced.
